# SHADOW CORRIDOR
### A First-Person Horror Game Built with HTML, CSS, and JavaScript

![Game Screenshot](screenshot.png)

## 🎮 Overview
Shadow Corridor is a psychological horror game where players navigate through dark corridors while managing their sanity, flashlight battery, and avoiding supernatural entities.

## ✨ Features
- **First-person perspective** with mouse look controls
- **Dynamic lighting system** with flashlight mechanics
- **Sanity system** that affects gameplay
- **Enemy AI** with patrol and chase behaviors
- **Inventory system** for collecting items
- **Interactive objects** (notes, batteries, medkits, keys)
- **Multiple game states** (menu, pause, game over)
- **Responsive UI** with health, sanity, and battery indicators

## 🎯 How to Play

### Controls
- **WASD / Arrow Keys** - Move
- **Mouse** - Look around
- **F** - Toggle flashlight
- **E** - Interact with objects
- **ESC** - Pause game

### Objectives
1. **Survive** - Manage your health and sanity
2. **Explore** - Find notes to uncover the story
3. **Collect** - Gather batteries and medkits
4. **Escape** - Find the exit key
